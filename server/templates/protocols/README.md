# Protocol Excel Templates

Place your protocol Excel template files here:

- MF6pt2.xlsx - MF 6.2 protocol template
- MF5pt2.xlsx - MF 5.2 protocol template
- FTire.xlsx - FTire protocol template
- CDTire.xlsx - CDTire protocol template
- Custom.xlsx - Custom protocol template

These templates are used when creating new projects and are copied to the protocol directory.

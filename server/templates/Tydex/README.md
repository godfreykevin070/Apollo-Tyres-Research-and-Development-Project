# Tydex Templates

Place your Tydex template files (.tdx) in protocol-specific subdirectories:

- MF62/
- MF52/
- FTire/
- CDTire/
- Custom/

Each template should follow the Tydex format with placeholders that will be replaced with simulation data.

Example placeholder names:
- `INCLANGL` - Inclination angle (radians)
- `LONGSLIP` - Longitudinal slip ratio
- `SLIPANGL` - Slip angle (radians)
- `LOAD` - Load (N)
- `PRESSURE` - Inflation pressure (Pa)
- `VELOCITY` - Test velocity (m/s)
